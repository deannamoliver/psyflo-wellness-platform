# @feelwell/backend

feelwell backend built with [Hono](https://hono.dev/) and uses [Postgres](https://www.mongodb.com/) as the database.

## Prerequisites

- [Bun](https://bun.sh/): Package manager
- [Docker](https://www.docker.com/): Build container images

## Environment Variables

- `DATABASE_URI`: The URI for connecting to the Postgres instance
