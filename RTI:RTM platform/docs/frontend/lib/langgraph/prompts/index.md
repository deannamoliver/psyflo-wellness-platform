# prompts/index.ts

Centralized export file for all LLM prompts. Re-exports prompts from `risk-detection.ts`, `handover.ts`, `screening.ts`, `system.ts`, and `utilities.ts` for convenient single-import access.

**Exports:** `RISK_DETECTION_PROMPT`, `HANDOVER_PROMPT`, `getBaseAgentSystemPrompt`, `getConversationSummaryPrompt`, clarification prompts, and parsing utilities
