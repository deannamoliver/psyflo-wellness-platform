# risk-protocol/conversation-state.ts

Defines the shared `ConversationState` annotation used across all graph nodes and subgraphs. Contains fields for messages, risk detection results, alert tracking, CSSR screening state, protocol control flags, and metadata. Also exports type helpers and `DEFAULT_CONVERSATION_STATE` initializer.

**Used by:** All graph files (`main-graph.ts`, `base-agent.ts`, `risk-router.ts`, subgraphs)
