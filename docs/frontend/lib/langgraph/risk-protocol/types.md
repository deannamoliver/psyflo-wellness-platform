# risk-protocol/types.ts

TypeScript type definitions for the risk protocol system. Defines `CSSRState` (tracking Q1-Q6 answers), `ClarificationResponses`, `PatternType`, alert result types (`CreateAlertResult`, `CreateChatAlertResult`, `UpdateChatAlertResult`), and other protocol-specific interfaces.

**Used by:** All risk protocol files, `tools/alert-tools.ts`
